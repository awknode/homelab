#### Download Clients

Settings -> Dowload clients

###### rtorrent
- hostname rtorrent
- port 3000
- Url Base /
- username/password set in rtorrent
  
###### nzbget
- hostname nzbget
- port 6789
- username/password set in nzbget
- Category: Movies
- full sync
